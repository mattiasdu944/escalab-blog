React Js v18

En nuestra última publicación, compartimos instrucciones paso a paso para actualizar su aplicación a React 18. En esta publicación, daremos una descripción general de las novedades en React 18 y lo que significa para el futuro.

Nuestra última versión principal incluye mejoras listas para usar, como procesamiento por lotes automático, nuevos API como startTransition y transmisión del lado del servidor con soporte para Suspense.

Muchas de las funciones de React 18 se basan en nuestro nuevo renderizador concurrente, un cambio detrás de escena que desbloquea nuevas y poderosas capacidades. Concurrent React es opt-in — solo está habilitado cuando usa una función simultánea — pero creemos que tendrá un gran impacto en la forma en que las personas construyen aplicaciones.

Pasamos años investigando y desarrollando soporte para la concurrencia en React, y nos hemos esforzado mucho para proporcionar una ruta de adopción gradual para los usuarios existentes.

El verano pasado, formamos el Grupo de trabajo de React 18 para recopilar comentarios de expertos de la comunidad y garantizar una experiencia de actualización fluida para todo el ecosistema de React.
